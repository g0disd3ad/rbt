# Red-Black Tree Dictionary

Консольное приложение и REST API сервис для работы с англо-русским словарем. 

Этот проект — адаптация моей курсовой работы за 1 курс программной инженерии по предмету Структуры данных. Изначально структура данных (Красно-черное дерево) была реализована на C++. В рамках подготовки к отбору я переписала алгоритм на Go, добавила дженерики, интерфейсы, REST API и хранение данных в PostgreSQL.

## Стек технологий
* **Язык:** Go (только стандартная библиотека `net/http`)
* **База данных:** PostgreSQL + драйвер `github.com/lib/pq`
* **Архитектура:** Разделение на слой хранения (адаптер для дерева / БД) и слой бизнес-логики.

## Функционал
* Быстрый поиск, добавление и удаление слов в памяти за `O(log N)` благодаря Красно-черному дереву.
* Валидация свойств дерева (проверка высоты и цвета узлов).
* Синхронизация данных с базой PostgreSQL.
* Импорт и экспорт словарей в обычные `.txt` файлы.
* Интерактивное консольное меню (CLI).
* Параллельно работающий HTTP-сервер для обработки REST-запросов.
